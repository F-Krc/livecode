# Hilfe

**Anweisungen**
* Erhalte Hilfe für `pwd`.
  $ pwd --help
  $ man pwd

* Erhalte Hilfe für `cd`.
  $ cd --help

* Erhalte Hilfe für `ls`
  $ ls --help
  $ man ls

* Lies die Handbuchseite für `pwd`.
  $ man pwd
* Lies die Handbuchseite für `cd`.
  $ man cd
* Lies die Handbuchseite für `ls`.
  $ man ls

* Welche Taste beendet das Handbuch? 
  'q'
